¿Qué es Finder?

Finder es una aplicación web diseñada para actuar como una alerta comunitaria de animales. Su objetivo es facilitar la localización y recuperación de mascotas perdidas mediante la creación de una plataforma centralizada donde los usuarios pueden reportar la pérdida de sus mascotas y recibir notificaciones si alguien las encuentra.
